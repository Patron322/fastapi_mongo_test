# Для запуска проекта требуется:
## 1. Склонировать репозиторий
## 2. Добавить в корневой каталог файл .env:

- MONGO_HOST=mongodb
- MONGO_PORT=27017

## 3. Выполнить команду:

```
make run
```

## 4. Для наполнения тестовыми данными после запуска нужно запустить команду:

```
make insert_data
```
